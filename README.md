# 2文字マスター — Japanese 2-Character Vocabulary Drill

424 two-character hiragana words. Flashcard drilling with Know/Shaky/Don't Know grading, gap-focused practice, and speed runs. Progress saved to localStorage.

## Deploy to Vercel (5 minutes)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Sign up with GitHub (free)
3. Click "Add New Project" → Import your GitHub repo
4. Framework: **Vite** (auto-detected)
5. Click **Deploy**
6. Your site is live at `your-project.vercel.app`

Any future `git push` to main auto-deploys.

## Local Dev

```bash
npm install
npm run dev
```
